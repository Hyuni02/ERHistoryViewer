### 1. [<u>*이터널리턴*</u>](https://playeternalreturn.com/main?hl=ko-KR) :link: 이란?
![이터널리턴배경](/readmeimgs/이터널리턴배경.png)<br>
- 넵튠의 자회사 님블뉴런에서 만든 쿼터뷰 배틀로얄
- 게임당 최대 24명의 플레이어가 참여하여 파밍한 물품들을
이용해 장비와 음식을 만들고 활용하는 배틀로얄, 다양한 전략으로 
최후의 1팀을 가려내는 게임
